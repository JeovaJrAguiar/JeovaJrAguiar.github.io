Componente Label com JAVA FX

Label pode ser traduzido para o português como "Rótulo" ou mesmo "Legenda". Para criar um label utilizando JAVA FX, basta importar e instanciar o objeto Label.
Veja o exemplo:
Label label = new Label("My Label");
Note que a string passada como parâmetro para o construtor, é a que será o rotulo do label.


Para tornar o Label visível, você deve adicioná-lo ao gráfico de cena. Isso significa adicioná-lo a um objeto Scene ou como filho de um layout anexado a um objeto Scene. Aqui está um exemplo que anexa um label JavaFX ao gráfico de cena:

JAVA FX
Bem, como o JavaFX não possui um instalador, o modo de usar é adicionando seus arquivos(no caso os de biblioteca) na área de trabalho de qualquer projeto dentro do Eclipse. E depois importar no projeto através de um import.


Abra o Eclipse e crie uma nova biblioteca de usuário com Eclipse -> Preferencias -> Java -> Build Path -> Biblioteca de usuário -> Novo.

Nomeie a bliblioteca com "JAVAFX18".




Clique em "Adicionar JARs externos" na barra lateral direita.
Selecione de dentro dos arquivos de você baixou no site da JavaFX.
Dentro da pasta Lib, todos dos arquivos com extenção ".jar"




Clique em "Aplicar e fechar".




Agora temos nosso ambiente com as ferramentas descessárias para ultilizar o JAVAFX.
